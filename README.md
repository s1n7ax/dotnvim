## Features

- Lazy load all plugins (except `startup-time`)
- Tree Sitter syntax highlighting
- Pre-configured for various programming languages and documents
  - Java
  - Python
  - JavaScript
  - Typescript
  - HTML
  - CSS
  - JSON
  - Markdown

## WIP

  - Python current file debugging is available
